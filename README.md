# Port Scanner GUI

Java приложение с графическим интерфейсом для сканирования открытых портов.

![Screenshot](GUI.png) <!-- Добавьте скриншот -->

## Особенности
- Сканирование всех 65535 портов
- Многопоточная реализация (100 потоков)
- Визуализация прогресса
- Простой графический интерфейс

## Как использовать
1. Введите адрес хоста (например `localhost` или `127.0.0.1`)
2. Нажмите "Начать сканирование"
3. Открытые порты будут отображаться в окне результатов

## Сборка и запуск
```bash
javac PortScannerGUI.java
java PortScannerGUI
```
